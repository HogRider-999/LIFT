## 📱 如何在 iPad/iPhone 上「安裝」？

為了獲得最佳體驗，建議將網頁加入主畫面，這樣就不會有瀏覽器網址列：

1. 使用 **Safari** 瀏覽器開啟上面的網址。
2. 點擊螢幕上方的 **「分享」** 圖示 (正方形帶箭頭 ⬆️)。
3. 向下捲動並選擇 **「加入主畫面 (Add to Home Screen)」**。
4. 點擊 **「新增」**，LIFT 就會出現在你的桌面上了！
